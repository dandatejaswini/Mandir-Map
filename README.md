# Mandir Map

Mandir Map is a web application that helps users explore temple locations, view temple details, and plan travel itineraries.  
The platform provides information about different temples and allows users to create local travel plans through a simple and responsive interface.

## Features
- Explore temples and view detailed information
- Select temples and create travel itineraries
- Local itinerary planner
- Multi-page navigation for temple details and planning

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase
